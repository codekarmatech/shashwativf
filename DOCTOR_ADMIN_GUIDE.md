# Shashwat IVF Admin Guide

This guide is for the clinic team to update website content from the admin panel without technical knowledge.

## Admin Login

Open the admin panel here:

[Open Admin Panel](https://api.shashwativf.com/admin/)

If you already have a username and password, log in there.

## Before You Start

- Changes made in the admin panel affect the website content.
- Some changes appear immediately.
- If something does not update after saving, refresh the page once.
- If the website layout, colors, buttons, or section design needs to change, that is not an admin change. That needs developer help.

Open:
[Clinic Information](https://api.shashwativf.com/admin/contact/clinicinfo/)

- Clinic name
- Tagline
- Short clinic description
- Full address
- Front desk phone
- Appointment phone
- Emergency phone

- Appointment email

This section controls website contact details shown in areas like:

- footer

- homepage contact sections

Open:
[Services](https://api.shashwativf.com/admin/services/service/)

Important:

## What You Cannot Control From Admin

The following items are not currently editable from the admin panel:

- Website layout and design
- Colors, fonts, button styles, spacing
- Home page section design
- Navigation menu structure
- New pages or page URLs
- Contact form behavior
- Technical SEO setup
- Server, hosting, domain, SSL, deployment
- Timeline section content on About pages
- Foundation / social impact static content
- "Why Choose Shashwat IVF" static reasons

If you want to change any of the above, you need developer help.

## Safe Way To Edit Content

For any content update:

1. Open the correct admin section from the links above.
2. Click the item you want to edit.
3. Change only the needed fields.
4. Click `Save`.
5. Open the website and verify the change.

## Important Publishing Rules

- `Published` means visible on the website.
- `Active` means profile is usable on the website.
- `Featured` means it may be highlighted on the home page or top sections.
- `Order` means smaller numbers usually appear first.

## Best Practice For Images

- Use clear, high-quality images.
- Prefer landscape images for banners and blog covers.
- Prefer square or portrait images for doctor/team photos.
- Avoid blurry WhatsApp-forwarded images when possible.

## If Something Does Not Show On The Website

Check these first:

- Is `Published` turned on?
- Is `Active` turned on?
- Did you click `Save`?
- Did you refresh the website page?
- Is the item in the correct category?
- Is the `Order` field causing it to appear lower on the page?

If it still does not appear, contact the developer.

## When To Contact Developer Support

Please ask for developer help if you need:

- a new section on the website
- a design change
- a new page
- a change in navigation/menu
- WhatsApp button behavior changes
- booking form changes
- new admin features
- content updates in static sections listed above
